# papers

This directory will contain research notes, methodology papers, and evaluation reports that document the scientific basis for the zbal attribution and false-flag models.

## Planned publications

- **Attribution Methodology** — Describes the Dempster-Shafer lattice approach and its advantages over point-estimate attribution models
- **False Flag Detection** — Presents the coherence scoring approach and evaluation against synthetic deception datasets
- **Evaluation Report** — Backtesting results on labeled historical incidents with precision, recall, and calibration metrics
